Game Of Life

https://en.wikipedia.org/wiki/Conway%27s_Game_of_Life

Step-by-Step Installation Instructions

  1. Install pygame, if not yet installed
  2. Download gameOfLife.py
  3. Execute command: python3 gameOfLife.py --run
  4. Enjoy

If you want to play with the game:

  1. Star game without --run parameter, you can also use --help: python3 gameOfLife.py
    • Mouse can be used to add alive cells
    • Press 'r' to randomize alive cells
    • Press 'n' to go next step
    • Press 's' to start gameloop
    • Press 't' to stop gameloop
    • Press 'c' to clear all cells
  2. Enjoy
