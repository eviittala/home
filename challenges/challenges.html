How to setup unit test environment in Windows and with MINGW

  1. mkdir test
  2. cd test
  3. git clone https://github.com/google/googletest.git
  4. add these files to your test-folder
    • CMakeLists.txt
    • Test.cpp
  5. mkdir build
  6. cd build
  7. cmake -G "MSYS Makefiles" ..
  8. cmake --build .
  9. Test.exe

It is possible to make 2€ in the following way: 1x1€ + 1x50c + 2x20c + 1x5c + 1x2c + 3x1c. Write a program to find different ways where 2€ can be made using any number of coins.

6. Linked List

A) Singly linked list
Write Linked List program which pushes values to back and front. Class interface should have functions push_back, push_front and print_list.

Input: 

LinkedList lList;
for (uint64_t i : {5, 6, 7, 8, 9, 10})
{
    lList.push_back(i);
}
lList.push_front(99);
lList.push_front(98);

lList.print_list();
98 -> 99 -> 5 -> 6 -> 7-> 8 -> 9 -> 10
